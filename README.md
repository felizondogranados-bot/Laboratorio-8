# Laboratorio 8

Aplicación desarrollada utilizando:

- HTML
- CSS
- JavaScript
- Fetch API
- Async/Await

## Descripción

El proyecto consume la API de TVMaze para mostrar:

- Información de una serie
- Rating
- Temporadas
- Episodios

Los episodios se renderizan dinámicamente en el DOM y utilizan colores según la valoración de cada episodio.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript ES6
- Fetch API
- TVMaze API
